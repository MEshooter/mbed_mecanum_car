## 本地调试方法

clone 所有文件至同一文件夹中，导入 mbed OS 库，再通过 Mbed Studio 编译运行

## 单命令

此类指令以 `#` 开头，形如 `# A`

`U/u`：开始/停止前进

`D/d`：开始/停止后退

`L/l`：开始/停止左移

`R/r`：开始/停止右移

`B/b`：开始/停止逆时针旋转

`E/e`：开始/停止顺时针旋转

## 多参数命令

此类指令以 `:` 开头，形如 `: V 0.71 0.71`

`V v_x v_y`：发送速度向量 $\mathbf{v}=(v_x, v_y)$ ，参数为浮点数

`SM JK/BT`：切换到摇杆/按键模式

`SPD speed`：调整基础速度为 `speed`，参数为整数类型

`SVO UD/LR level`: 调整控制上下/左右方向的舵机至指定角度，`level` 为整数，表示 $-90^\circ$ 至 $~90^\circ$ 到 $-9$ 至 $9$ 的线性映射
